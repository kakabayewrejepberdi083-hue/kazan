# Kazan City Website

This project is a simple multi-page website about Kazan, the capital of the Republic of Tatarstan, Russia.

## 📌 About the Project
The website presents information about Kazan using videos and images.  
It includes short drone videos (10–20 seconds) and around 70 photos collected from the internet.

The goal of this project is to showcase the beauty, architecture, and atmosphere of Kazan.

##  Features
- Multi-page structure (5 pages)
- Navigation menu
- Video gallery (drone footage)
- Photo gallery
- Informational text about the city
- Responsive layout (if sen yaptıysan ekle, yapmadıysan sil)

##  Technologies Used
- HTML5
- CSS3

## 📂 Pages
- Home page
- Kremlin
- Bridge
- Family Center
- KFU University
- National Library

## 🎥 Media
- Drone videos (short clips)
- Photography (aerial and ground images)

> Note: All media used in this project are for educational purposes.
